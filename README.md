# Ficha de Personagem 4C

Este projeto é uma ficha digital para o sistema de RPG **4C**, desenvolvida por Ismael “Mael” Spinelli. A ficha é responsiva, personalizável e pronta para ser usada online ou impressa.

## 📌 Recursos

- Interface clara e escura, responsiva
- Alternância de idioma: Português / English
- Cálculo automático de atributos como Vitalidade, Vontade e Ficha de Combate
- Exportação para `.txt`
- Impressão estilizada em uma única página com logotipo
- Rodapé e layout otimizado para mesa ou PDF

## 🚀 Como usar

1. Acesse: **https://mael-retro-rpg.github.io/4C-CS/**
2. Preencha os campos diretamente no navegador
3. Clique nos ícones para:
   - 📄 Exportar como `.txt`
   - 🖨️ Gerar uma ficha de impressão com todos os dados formatados

## 🧾 Arquivos

- `index.html`: página principal com a ficha interativa
- `logo-ficha.png`: logotipo da ficha principal
- `4CP.png`: logotipo da versão de impressão
- `README.md`: este arquivo
